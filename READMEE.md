# OS Jackfruit - Multi Container Runtime

## Overview

This project implements a lightweight container runtime using Linux namespaces and kernel modules.

## Features

* Container isolation using clone()
* CLI-based container management
* Logging using producer-consumer model
* Kernel module for memory monitoring
* Scheduler experiments
